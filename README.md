## Getting Started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

## Project Structure

```
app/                    # Demo screens for each component
components/             # Reusable components
  ├── animatedText/     # Text animation effects
  ├── AnimatedSlider.tsx
  ├── QuickMenuButton.tsx
```
